Docker是个好东西

<!--more-->

```
##搜索镜像
docker search 镜像名字

##下载镜像
docker pull 镜像名字全称

##启动 Docker 进程
sudo systemctl start docker

##列出所有正在运行的容器
docker ps

##列出所有的镜像
docker image ls

##暂停容器
docker stop <name>

##删除容器
docker rm <name>

```
